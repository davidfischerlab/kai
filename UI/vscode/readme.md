# Kai Agent for VS Code

AI-powered bioinformatics analysis assistant for Jupyter notebooks in VS Code.

## Configuration

Open VS Code settings (`Ctrl+,`) and search for "kai agent":

- `kai_agent.pythonPath`: Path to Python with kai-agent installed
- `kai_agent.projectPath`: Path to kai-agent project directory (leave empty for auto-detection)
- `kai_agent.ollamaApiKey`: Ollama API key (optional - can use OLLAMA_API_KEY environment variable)

## Usage

### 1. Open a Jupyter Notebook
Open any `.ipynb` file in VS Code.

### 2. Use Chat Interface
Click the Kai icon in the sidebar to open the chat interface:
- Ask questions about your analysis
- Request code generation
- Get help with errors
- Enable autonomous mode for multi-step workflows
